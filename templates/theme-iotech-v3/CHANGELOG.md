# @iotech/theme-iotech

### Initial Release

- Homepage with hero, services grid, blog preview, CTA
- Blog archive with pagination and single post view (PortableText)
- Services page with individual service detail
- À propos static page
- Contact form page

- RSS feed at `/rss.xml`
- Custom 404 page
- Dark theme with IØTech brand colors (Syne + JetBrains Mono)
- Seed data validated against EmDash CLI v1
